KI-Chatbot Lehrtool

Ein ChatGPT-, Gemini- und Claude-artiges Lernfrontend.

Was passiert technisch?

  1. Das Frontend sammelt Anbieter, API-Key, Modell, System Prompt und Chatverlauf.
  2. Der Browser sendet diese Daten per JSON an /api/chat.
  3. Python übersetzt den Verlauf in das Format des gewählten Anbieters.
  4. Die Antwort kommt zurück und wird als neue Chatblase angezeigt.
